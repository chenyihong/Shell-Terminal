# Shell-Terminal

```
CTRL+ALT+T                        //打开终端

id                                //用于查看USERID

ls                                //用于查看路径

man ls                            //查看ls的手册

touch *.txt                       //创建新的txt文件

ls -al 文件名.文件格式             //查看文件的权限

chgrp groupName 文件名.文件格式    //改变文件的组群

chmod g+w 文件名.文件格式          //改变文件的权限(使组群获得写入的权限)

chmod o-r 文件名.文件格式          //改变文件的权限(删除所有其他组群的读取权限)

ps auxwww                         //类似于任务管理器(查看USERID和GROUPID)

kill [PID]                        //杀死一个指定PID的进程

pwd                               //print working directory 打印现在所在的路径

cd ..                             //返回上一层路径

cd [path]                         //进入指定文件夹(路径)

mkdir [路径/文件夹名字]            //创建路径

rmdir [路径/文件夹名字]            //删除路径

rm  [文件名字.文件格式]            //删除文件

cp [被复制的文件路径] [复制的目的]  //复制文件

ln -s [文件路径] [文件路径]        //用于创建文件快捷路径(如果被复制文件被删除,则链接的文件不可用,提示找不到文件)

ln [文件路径] [文件路径]           //类似于cp的作用(如果被复制文件被删除,但是链接的文件依旧可用)

cat [文件名.文件格式]              //查看文件内容

quota -A                          //查看可用空间

CTRL+F1/F2/F3                     //用于切换图形窗口/终端窗口
```
